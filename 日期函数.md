#### 年月日时分秒获取

一、得到日期的年
```js
let date = new Date(); 
let year = date.getFullYear();
```

二、得到日期的月
```js
let month=date.getMonth()+1;
// 拿到的 month 值一般是从0开始的，所以需要+1
```
三、得到日期的日(具体几号)
```js
let date = date.getDate();
```
四、得到星期几
```js
let day = date.getDay();
```
五、得到小时
```js
let hour = date.getHour();
```
六、得到分钟
```js
let minutes = date.getMinutes();
```
